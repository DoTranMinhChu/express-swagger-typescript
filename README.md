# Installing

For the latest stable version, run:

## npm

```sh
# TypeScript later
npm install express-swagger-typescript swagger-ui-express

```

## yarn

```sh
# TypeScript later
yarn add express-swagger-typescript swagger-ui-express

```

# Usage

Set tsconfig.json:

```json
{
    "compilerOptions": {
        "experimentalDecorators": true,
        "emitDecoratorMetadata": true
    }
}
```
